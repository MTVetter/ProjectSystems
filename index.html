LA DOTD Projects

This application was created by DOTD GIS in August 2018.

All of the project information was created by DOTD in September of 2017. The project data is currently being updated annually with the intent for it to be updated daily in the future.

The urbanized area value is populated from the Urbanized Area feature that is maintained by DOTD GIS. The current feature was last revised on 09/18/2014 and are based on the 2010 US Census data and agreements made between DOTD and local metropolitan areas.

The parish layer was created using original linework that was manually digitized using USGS Quadrangle Maps along with legal parish boundary descriptions.

The functional classification values is populated from the Functional Classification feature which is the 2014 final Functional Classification of all public roads in Louisiana based on the 2010 US Census data.

The routes displayed in this map are one of two LRS Route Networks that the department manages. LRSID routes is the department's routes using the Control Section/Log Mile Linear Referencing Method. LRSID routes is being managed and updated daily.

This is a step by step guide to help add a prosed project.

The side panel allows you to enter information about the project before drawing it.
You can also allow the system to automatically populate the majority of the fields for you.

When drawing the project, you need to be zoomed in to verify that you're drawing on the roads.
A zoom level like the following screenshot should be good:



After drawing the project, you will see a confirmation box like below:
This box will only display if you put a project on a non-divided roadway.
Click OK if you want to use the route mentioned in the box; otherwise click Cancel.

After clicking OK or Cancel, you'll notice that the attribute fields have been filled in.
You can change any of the values in the box or you can click Clear Fields to start all over.

Once you have the correct information in the attribute boxes, click Submit.
